This file is created because it carries basic git commands !

  1. Git Clone :
    Syntax = "git clone url(url of git repositary)"+ Use : This will download the git repositary from git hub .
  2. Git add : Syntax = " git add '[file name]'" Use : This will tell the git to add this file on the server of the GitHub
  3. Git Commit : Syntax " git commit - m "message"" Use : This will save the change that you did and the message will be saved
  4. Git Status : Syntax : "git status" Use : This will give me the report what is currently happening in my repositary
  5. Git Push : Syntax : "git push" Use : This command will push all my commits and add-ons to the git hub main server
  6. Git Commit -am : Syntax : "git commit -am "message"" Use : This command will work by combining both the git add and git commit command
  7. Git Pull : Syntax : "git pull " Use : This command is used to pull the latest version of the file from the git hub server .
