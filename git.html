This file is created because it carries basic git commands

  1. Git Clone :
    Syntax = "git clone url(url of git repositary)"+ Use : This will download the git repositary from git hub .
  2. Git add : Syntax = " git add '[file name]'" Use : This will tell the git to add this file on the server of the GitHub
  3. Git Commit : Syntax " git commit - m "message"" Use : This will save the change that you did and the message will be saved
